# Diffusion Transformer (DiT) from Scratch вҖ” CIFAR-10

A from-scratch **Micro Diffusion Transformer (DiT)** trained in pixel space on CIFAR-10 (32Г—32Г—3). No pre-trained VAE вҖ” pure pixel-level diffusion with a transformer backbone.

| Spec | Value |
|---|---|
| Dataset | CIFAR-10 (50k images, 32Г—32Г—3) |
| Patch size | 4Г—4 вҶ’ **64 tokens** |
| Hidden dim | 384 |
| Attention heads | 6 |
| Transformer blocks | 6 (adaLN-Zero) |
| Parameters | ~30M |
| Diffusion steps | T = 1000 (linear ОІ schedule) |
| Loss | MSE (Оө-prediction) |

## Architecture Overview

```mermaid
flowchart TD
    subgraph Input
        A[" Noisy Image\n(B, 3, 32, 32)"]
        T[" Timestep t"]
    end

    subgraph Patchify
        A --> P["Conv2d(3вҶ’384, k=4, s=4)\nвҶ’ (B, 64, 384)"]
        P --> PE["+ 2D Sinusoidal\nPositional Embedding"]
    end

    subgraph Conditioning
        T --> SE["Sinusoidal Encoding"]
        SE --> MLP["MLP: LinearвҶ’SiLUвҶ’Linear\nвҶ’ c вҲҲ в„қВівҒёвҒҙ"]
    end

    subgraph Transformer["6Г— adaLN-Zero Transformer Blocks"]
        PE --> BLOCK
        MLP --> MOD["c вҶ’ MLP вҶ’ ОівӮҒ,ОІвӮҒ,ОұвӮҒ,ОівӮӮ,ОІвӮӮ,ОұвӮӮ"]

        subgraph BLOCK["Each Block"]
            direction TB
            IN["Tokens x"] --> LN1["LayerNorm (no affine)"]
            LN1 --> SC1["Scale & Shift (ОівӮҒ, ОІвӮҒ)"]
            SC1 --> ATTN["Multi-Head Self-Attention"]
            ATTN --> G1["Г— ОұвӮҒ (gate, init=0)"]
            G1 --> R1["+ x (residual)"]
            IN --> R1
            R1 --> LN2["LayerNorm (no affine)"]
            LN2 --> SC2["Scale & Shift (ОівӮӮ, ОІвӮӮ)"]
            SC2 --> FFN["Feed-Forward Network\nLinearвҶ’GELUвҶ’Linear"]
            FFN --> G2["Г— ОұвӮӮ (gate, init=0)"]
            G2 --> R2["+ residual"]
            R1 --> R2
        end

        MOD --> SC1
        MOD --> SC2
    end

    subgraph Decode["Final Layer & Unpatchify"]
        R2 --> FL["adaLN + Linear(384вҶ’48)"]
        FL --> UP["Reshape (B,64,48)\nвҶ’ (B,3,32,32)"]
    end

    UP --> OUT[" Predicted Noise Оө_Оё\n(B, 3, 32, 32)"]
```

### Training Loop

```mermaid
flowchart LR
    X0["xвӮҖ (clean)"] --> QS["q_sample:\nx_t = вҲҡбҫұ_tВ·xвӮҖ + вҲҡ(1-бҫұ_t)В·Оө"]
    E["Оө ~ N(0,I)"] --> QS
    T["t ~ U[1,T]"] --> QS
    QS --> DiT["DiT(x_t, t) вҶ’ Оө_Оё"]
    DiT --> L["L = MSE(Оө, Оө_Оё)"]
    L --> BP["Backprop\nAdamW"]
    BP -.-> X0
```

### Sampling (Reverse Diffusion)

```mermaid
flowchart LR
    N["x_T ~ N(0,I)"] --> D1["Denoise\nt=999"]
    D1 --> D2["Denoise\nt=998"]
    D2 --> DOTS["вӢҜ"]
    DOTS --> D3["Denoise\nt=1"]
    D3 --> D4["Denoise\nt=0"]
    D4 --> IMG[" Generated\nImage xвӮҖ"]
```

$$x_{t-1} = \frac{1}{\sqrt{\alpha_t}} \left( x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}} \cdot \varepsilon_\theta(x_t, t) \right) + \sigma_t \cdot z$$

## How to Run

1. Open `DiT.ipynb` in **Google Colab**
2. Set runtime to **T4 GPU** (`Runtime вҶ’ Change runtime type вҶ’ T4`)
3. Click **Run All**
4. Training runs for ~80 epochs with checkpoints every 10
5. After training, sampling cells generate a 4Г—4 grid of images
